About Event Manager

Event Manager is a simple web application designed to help users create, manage, and view upcoming events. You can easily add events with a name, description, and date/time, and they’ll be saved in your browser.

The app is built with modern web technologies including HTML5, CSS3, and JavaScript. It stores your events locally so you can manage them anytime without needing an account or internet connection.

Whether you’re planning meetings, webinars, or personal reminders — Event Manager keeps everything organized in one place.
